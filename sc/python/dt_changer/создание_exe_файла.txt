Один из способов создания .exe файла:

1. Для создания .exe необходимо иметь pip. Известный способ использования pip - установка python 3+; 
pip будет лежать по пути Python\Scripts
2. С его помощью необходимо установить pyinstaller:
> pip install pyinstaller
3. С консоли перейти в директорию, где лежит ch.py
4. Создать .exe файл:
> pyinstaller -F ch.py
Файл лежит в директории dist. Остальное можно удалить
